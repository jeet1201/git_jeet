Yelp Web Mashup (REST API, JSON, JavaScript, HTML5, AJAX)		       		                                            
●	Created a web mashup application using Google Maps and Yelp Fusion API using JavaScript and AJAX 
●	After the Google Map is centered to a geographical area and cuisine category is given in the input, the application finds the best matches inside the map area
●	It also marks the result locations on map and displays information (JSON) about restaurants in the webpage
